# Solidity Kickstarter

A blockchain powered Kickstarter built with NextJS, TS & Solidity.

## Installation

Use [yarn](https://classic.yarnpkg.com/en/docs/install#mac-stable) to install this POC.

```bash
yarn
```

## Usage

```
yarn dev
```
